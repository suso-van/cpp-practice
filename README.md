# C++ Programming Practice

This repository contains my **C++ programming practice codes**, focusing on high-performance programming, object-oriented concepts, and algorithmic problem solving.

The programs cover both **procedural** and **object-oriented** aspects of C++, useful for competitive programming, system-level understanding, and interviews.

---

## 📂 Contents

- C++ basics (syntax, input/output)
- Control statements and loops
- Functions and recursion
- Arrays, strings, and pointers
- Object-Oriented Programming (OOP)
- STL (Standard Template Library)
  - Vectors, maps, sets
  - Stacks, queues
- Algorithms and problem-solving
- Practice problems and logic building

---

## ▶️ How to Run

Compile and run a C++ program using:

```
g++ filename.cpp -o output
./output
```
